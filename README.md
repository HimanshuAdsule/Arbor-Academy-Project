# Arbor-Academy-Project
1.Employee Sales Analysis-
This project analyzes employee sales data using Excel. It includes data cleaning, pivot tables, and interactive dashboards to track sales performance, top employees, and regional trends. The objective is to generate actionable insights that help improve business decisions and overall sales efficiency.
2.Hospital Data Analysis -
This project focuses on hospital and patient data analysis using Excel. It covers data cleaning, KPI creation, and dashboard development to monitor patient details, health metrics, and hospital performance. The aim is to provide meaningful insights for better healthcare decision-making.
1.Data Analytics Portfolio -
This repository showcases multiple data analytics projects including employee sales and hospital data analysis. It highlights skills in Excel, dashboards, and data visualization. The projects focus on solving real-world problems and delivering actionable insights for business improvement.
